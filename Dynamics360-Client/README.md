# React + Vite + css

### Build a single-page, responsive website with the following features:

### To Run

Open Terminal => npm run dev

### Main task

Requirements

Layout

1. A top navigation bar containing the site logo and links to “Home,” “Events,” and “Contact."
2. A hero section with a short heading:

Discover Events Near You.

1. A section titled “Featured Events” that displays 3–5 sample events.

Each Event Card Should Include.

1. Event name
2. Date and time
3. Location
4. Short description
5. A “Register” button (no actual functionality required)

Design

1. Use a simple, clean UI (either Bootstrap or your own custom styling).
2. Ensure the page is fully responsive (mobile- and tablet-friendly).

Bonus (Optional)

1. Add a search bar that filters events by name.
2. Load event data dynamically using dummy JSON (you may simulate fetching with JavaScript).

### Future Improvements.

4. aDD fontawesome.
